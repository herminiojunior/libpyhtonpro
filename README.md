
# libpyhtonpro
Módulo para exemplificar construção de projetos Python no curso PyTools

Nesse curso é ensinado como contribuir com projetos de código aberto

**Para instalar:**

`pip install pipenv`

`pipenv install --dev`

**Para conferir qualidade de código:**

`pipenv run flake8`

**Tópicos a serem abordados:**

<li>Git</li>
<li>Virtualenv</li>
<li>Pip</li>
<li>Mock</li>
<li>Pipenv</li>
